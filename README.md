# Syndicore iOS

Swift client for the Syndicore API. See [syndicore-BE](https://github.com/LukovicNikola/syndicore-BE) for the backend.

## Contracts

The `SyndicoreContracts/` folder is auto-synced from `syndicore-BE` on every merge to `main`.
Do not edit files in that folder manually — push contract changes to the BE repo instead.
